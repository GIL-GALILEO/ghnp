Clipping

The Georgia Historic Newspapers site has a clipping option that will allow you to print or save a portion of a newspaper page. If you are viewing a page you would like to clip, zoom in to the portion of the page you are interested in and click the “Clip Visible Area” button on the top right corner of the image. A new tab will load with the portion of the newspaper page you selected. You will then be able to print, save, or bookmark this clipped page image. For more information on how to zoom in on an image, please see Page Viewing Help.

Printing

If you are interested in printing a portion of a newspaper page, use the clipping option as described above. Once you have selected the “Clip Visible Area” button, a new tab will load with the portion of the newspaper page you selected. You can print the page using the “Print This Page” button above the image. The printed page will include information on the title, date, and persistent link for the clipped newspaper page.

Saving

If you would like to save a portion of a newspaper page for personal use:

  1. Zoom in to the portion of the page you would like to save and click the “Clip Visible Area” button on the top right corner of the image, as described above.
  2. A new tab will load with the portion of the newspaper page you selected.
  3. You can then right click the clipped newspaper page image using your mouse.
  4. From the right click menu, you can select the “save image as…” option. This will then allow you to save the newspaper page image as a JPEG in the folder of your choosing.

You can also save the newspaper page images as a bookmarked link.

  1. Once you have clipped the newspaper page, select the “bookmark” option in your browser menu (this is often located in the upper right corner of the browser).
  2. The URL will persistently link you to the image you have clipped.
